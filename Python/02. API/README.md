# 🌐 APIs — Consumo e Integração de Dados

Esta pasta reúne estudos voltados ao consumo, manipulação e integração de dados por meio de **APIs (Application Programming Interfaces)**.

O objetivo é compreender como sistemas se comunicam via HTTP, permitindo a coleta de dados externos e sua integração em fluxos analíticos e pipelines de dados.

Cada subpasta explora uma abordagem prática de consumo de APIs, utilizando diferentes bibliotecas e contextos de aplicação.

A organização foi estruturada por biblioteca, permitindo aprofundamento técnico progressivo.

## 🎯 Objetivo da Seção

- Compreender o funcionamento de APIs REST;
- Realizar requisições HTTP de forma estruturada;
- Integrar dados externos em análises e pipelines;
- Trabalhar com respostas em formato JSON;
- Desenvolver base para ingestão de dados em projetos de dados.

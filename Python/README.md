# 🐍 Python — Estudos por Biblioteca

Esta pasta reúne estudos práticos focados em bibliotecas específicas do ecossistema Python.

O objetivo é organizar o aprendizado por ferramenta, documentando conceitos fundamentais, aplicações práticas e experimentações realizadas durante o processo de estudo.

Cada subpasta representa uma biblioteca explorada de forma isolada, permitindo aprofundamento técnico direcionado.


## 🎯 Objetivo da Pasta

- Consolidar fundamentos técnicos por biblioteca;
- Explorar aplicações práticas além de exemplos básicos;
- Documentar testes e experimentações;
- Criar uma base estruturada para aplicações futuras em projetos maiores.


## 🧠 Metodologia

Para cada biblioteca estudada:

1. Leitura da documentação oficial;
2. Reprodução de exemplos básicos;
3. Desenvolvimento de pequenos scripts práticos;
4. Testes exploratórios;
5. Registro de aprendizados com base em projetos reais no README específico.
